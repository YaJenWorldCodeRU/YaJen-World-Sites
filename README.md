# YaJen-World-Sites

A collection of web applications and sites developed by YaJen World.

## Projects

### 🌤️ WeatherSite
A modern weather application with a sleek glass-morphism UI design.
- **File:** `WeatherSite.html`
- **Features:**
  - Responsive mobile-first design
  - Glass-morphism UI elements
  - Dark theme with gradient backgrounds
  - Touch-optimized interactions

### 🤖 YaJenGPT
An AI chatbot interface powered by TensorFlow.js with multiple model support.
- **Files:** 
  - `YaJenGPT.html` - Main chatbot interface
  - `YaJenGPT-Full.html` - Full-featured version
- **Features:**
  - Multiple AI model support
  - Language detection
  - WebGPU backend acceleration
  - Modern dark theme UI
  - Mobile-responsive design

## Usage

Simply open any `.html` file in a modern web browser to use the applications. No server setup required as these are standalone client-side applications.

## Requirements

- Modern web browser (Chrome, Firefox, Safari, Edge)
- JavaScript enabled
- Internet connection (for CDN resources and API calls)
